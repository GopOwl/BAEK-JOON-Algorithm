# 문제 주소 
https://www.acmicpc.net/problem/5622

## 문제 접근 방법 
문제에 영어에 따라 숫자가 적용이 되어 있는데 1이랑 0은 없어서 그런지 신경안써도 되는것 같음 그리고 숫자에 +1을 해줘야 걸리는 시간임 

### 새롭게 알게된 점 
문자열을 비교할수 있다는 것을 처음 알게됨 자바스크립트는 '사전’순으로 문자열을 비교한다. 사전 뒤쪽의 문자열은 사전 앞쪽의 문자열보다 크다고 판단된다.

예를 들어서 'Z' > 'A' Z가 더 뒤에 있기에 (사전상) true를 출력을 한다. 

근데 정확히는 사전순이 아니라 유니코드 순이라고 한다. 

자바스크립트는 대·소문자를 따진다는 것. 대문자 'A'와 소문자 'a'를 비교했을 때 소문자 'a'가 더 크다. 자바스크립트 내부에서 사용되는 인코딩 표인 유니코드에선 소문자가 대문자보다 더 큰 인덱스를 갖기 때문이라고 한다.
